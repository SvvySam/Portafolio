# Portafolio Profesional

Este es un portafolio web moderno y profesional creado con HTML5, CSS3 y JavaScript. Está diseñado para ser rápido, accesible y fácil de personalizar.

## Estructura del Proyecto

- **index.html**: El archivo principal que contiene toda la estructura y contenido del sitio.
- **styles/main.css**: Todos los estilos visuales. Aquí puedes cambiar colores, fuentes y espaciado.
- **scripts/app.js**: Lógica para el menú móvil y animaciones.

## Cómo Personalizar

1.  **Imágenes**:

    - Actualmente el sitio usa gradientes CSS como placeholders.
    - Para usar tus propias imágenes, guarda tus archivos en la carpeta `assets` (créala si no existe) y actualiza las etiquetas `<img>` o fondos CSS.

2.  **Textos**:

    - Abre `index.html` y busca los textos de ejemplo para reemplazarlos con tu información (Nombre, Biografía, Proyectos).

3.  **Colores**:
    - Abre `styles/main.css` y busca el bloque `:root` al principio del archivo. Cambia los colores hexagesimales (por ejemplo `--primary: #646cff;`) para ajustar el tema a tu gusto.

## Ejecutar Localmente

Simplemente abre el archivo `index.html` en tu navegador web favorito (Chrome, Edge, Firefox). ¡No se requiere instalación de servidores!
